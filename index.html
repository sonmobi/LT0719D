Đề bài tập thực hành

1. Hãy tạo thư mục dự án là: giao-dien-imag-lt0719d

2. Tạo file index.html trong thư mục và đẩy lên github.

3. Trên giao diện Repos của dự án vừa tạo trên github, bên dưới nút Star có link: Setting --> bấm vào link này để vào thiết lập riêng của Repos này. 

4. Cuộn màn hình xuống đến phần GitHub Pages, ở mục Source hãy chọn master --> website tự load lại trang, tiếp tục cuộn màn hình tới vị trí GitHub Pages  --> lúc này sẽ nhìn thấy có link mới, đây là link website tĩnh của bạn. VD: https://sonmobi.github.io/LT0719D/

Chú ý: Website hoạt động được thì phải có file index.html nhé. 

 
Yêu cầu viết bằng html, css, không sử dụng bootstrap.
Trả lời lại bài viết trên nhóm lớp với 2 link: Link code github và link github page.
